String
문자나 문자들을 나열한 것입니다.
값을 변경할 수 없으며 순서가 있습니다.
큰따옴표(" ")나 작은따옴표(' ')로 구분합니다.
>>> my_str = 'python'
>>> print(my_str)
python
>>> type(my_str)
<class 'str'>
문자열 안에 큰따옴표나 작은따옴표를 넣고 싶을 때는 서로 다른 따옴표를 함께 씁니다.
>>> print("Let's go!")
>>> print('I said, "Hello!"')
같은 따옴표를 세 번씩 쓰면 여러 줄을 표현할 수 있습니다.
>>> my_str = """버스
지하철
택시
"""
